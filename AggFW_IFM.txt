Best way is to update all AC14 controllers to the latest version which can be  find on the ifm  internet.

 

On the ifm home page please navigate to Service / Download / Industrial communication .

 

There you’ll find version 3.1.3. There is also an upgrade description put in the archive file.

 

You need a standard SD card for the update.

 

This version 3.1.3 corresponds to CODESYS version 3.5.4.2 and the AC14 package which you can also find in the download area.
